# Domain Notes

Species-level notes that may matter for labeling, augmentation, and interpretation.

## Bananaquit (`banana`, `Coereba flaveola`)
- Small nectar-feeding passerine with thin, high-pitched calls and songs.
- Vocal output can include short contact calls as well as more song-like phrases.
- Often occurs in edges, gardens, and scrubby habitat, so background noise can include other common edge species.
- Treat this class as potentially variable across context rather than a single fixed sound template.
